这是一个恋爱脑测试小程序，仅供学习使用。

该项目使用Taro搭建，使用Taro-UI。

### 启动项目
```bash
# 安装
npm install -g @tarojs/cli
   
# 初始化项目
taro init love-brain-test

# build之后在微信开发者工具中打开项目进行调试和测试
taro build --type weapp --watch

# 小程序预览
npm run dev:weapp
npm run dev:alipay
```

使用微信开发者工具打开dist文件夹，进行预览和调试。
